# Ledger Alchemy - Suggested Phase 1 Roadmap

## Goal
Turn the current MVP skeleton into a fun, polished, off-chain prototype before adding XRPL/NFT features.

## Phase 1A - Stabilise
- Confirm local setup works.
- Confirm backend starts.
- Confirm frontend starts.
- Confirm seed data loads.
- Confirm craft, inventory, and leaderboard endpoints work.
- Fix obvious bugs only.

## Phase 1B - Rebrand
- Rename user-facing copy from Infinite Craft XRPL to Ledger Alchemy.
- Update README.
- Update app title and page copy.
- Keep internal names if renaming them risks breaking things.

## Phase 1C - Improve Game Loop
- Add better starter elements.
- Add 30 to 50 handcrafted recipes.
- Add rarity tier labels.
- Add better result reveal.
- Add discovered combination history.

## Phase 1D - Improve UI
- Dark alchemy theme.
- Mobile-friendly crafting screen.
- Better inventory cards.
- Better leaderboard.
- Better empty/loading/error states.

## Phase 1E - Prepare for XRPL Later
- Keep wallet auth isolated.
- Document future wallet verification plan.
- Document future XLS-20 NFT minting plan.
- Do not add real transactions yet.
